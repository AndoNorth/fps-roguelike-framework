# Project: FPS Roguelike Framework

A fast-paced first-person roguelike framework built in Godot, focused on modular systems, procedural room generation, and combat experimentation.

## Overview

This project aims to establish a solid foundation for a replayable FPS roguelike experience.
Rather than building a full game upfront, the focus is on creating flexible systems that support:

* Procedural level generation using pre-built rooms
* Room-based combat encounters
* Expandable weapon systems (modern and historical firearms)
* Iterative gameplay-first development

The project is currently in early development and prioritizes mechanics over visuals.

## Core Direction

* **Genre:** First-Person Shooter / Roguelike
* **Engine:** Godot
* **Platform:** Web (WebGL build target)
* **Scope:** Singleplayer only (no multiplayer planned)

### Design Goals

* Fast, responsive FPS gameplay
* High replayability through procedural generation
* Modular, reusable systems (rooms, enemies, weapons)
* Rapid iteration on mechanics before committing to art direction

## Systems (Planned)

* **Room-Based Generation**

  * Pre-designed rooms assembled into randomized layouts
  * Inspired by roguelike dungeon structures

* **Combat Encounters**

  * Enemies spawned per room
  * Increasing difficulty over a run
  * Boss encounter at the end of a level/run

* **Weapons**

  * Mix of modern and classic firearms (e.g. SMGs, lever-action rifles)
  * Expandable system for adding new weapons and behaviors

## Current Focus

* Core player controller (movement, shooting feel)
* Basic enemy behavior and spawning
* Room system and procedural layout generation
* Weapon framework

Visuals, art style, and polish will be developed in parallel at a later stage.

## Project Structure

This repository serves as a **framework and foundation**, not a finished game.
Systems are being built to be extended, replaced, or reworked as development progresses.

## Roadmap

A detailed roadmap and design iterations will be maintained separately as the project evolves.
